# Bengali Text Analyzer

A professional Bengali text analysis tool combining classical NLP (via BNLP toolkit) with AI-powered insights (via Google Gemini API).

**Status:** 🚧 Under active development (v0.1.0)

## Features (Planned for v0.1)

- Basic statistics: word, sentence, character, paragraph counts
- Bengali-aware tokenization using BNLP toolkit
- Unicode normalization for consistent analysis
- Stop word analysis and vocabulary richness metrics
- AI-powered text summarization (short/medium/detailed)

## Tech Stack

- Python 3.12+
- Streamlit
- BNLP Toolkit
- Google Gemini API
